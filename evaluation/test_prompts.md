# Expanded Evaluation Test Set

This evaluation set is designed to test whether GreenPrompt appropriately influences format, tone, size, mode, detail level, and language.

---

## 1. Format Evaluation Prompts

**Table**  
- Create a table comparing the climates of Salt Lake City, Mumbai, and Paris.

**JSON**  
- Represent the pros and cons of online learning in a JSON structure.

**Single Paragraph**  
- Explain why photosynthesis is important for life on Earth in one concise paragraph.

**Bullet List**  
- List the main causes of the 2008 financial crisis.

**Multiple Paragraphs**  
- Describe how electric vehicles work and explain their environmental impact in multiple short paragraphs.
